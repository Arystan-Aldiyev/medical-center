Задача: Сайт для медицинского центра 
 
Администратор на своей странице может: 
Добавить нового пациента в систему предоставив соответствующие данные, пароль 
Добавить нового доктора в систему предоставив соответствующие данные, пароль 
Редактировать данные пациента (любое из вышеприведенных полей) 
Удалить пациента из системы 
 
For patients: 1) date of birth 2) IIN number 3) ID number 4) name, surname 6) Blood group 7) Emergency Contant Number 8) Contant number 9) Email (Optional) 10) Address 11) Marital Status 12) Registration Date 13) Any other optional details you find neccessary  
For doctors: 1) Date of birth 2) IIN number 3) ID number 4) name, surname, middlename 6) Contant number 7) Department ID 8) Specialization details ID 9) Experience in years 10) Photo of the doctor 11) Category (Highest, First, etc.) 12) Price of the appointment 13) Schedule details 14) Degree/education (MD, PhD, etc.) 15) Rating (a number in range from 0 to 10) 16) Address 17) Homepage url (optional)  
 
Доктор на своей странице может: 
Увидеть список всех своих проведенных и предстоящих консультаций 
Поисковая строка посредством которой производится поиск пациента (по ИИН) 
Увидеть историю визитов конкретного пациента: все прошлые визиты к врачам, их дату-время, название процедуры (пусть будет имя департамента) и принимавшего врача 
И историю лечения: название лекарства, дата начала, дата окончания, статус: активно или завершено. Со статусом активно могут быть изменены доктором 
Возможность распечатать страницу пациента 
 
Пациент на своей странице может: 
Изменить свои личные данные 
Увидеть свой план лечения, все свои прошлые  и предстоящие записи к врачам (идентично тому что на странице доктора) 
Записаться на прием 
 
Запись на прием: 
Страница должна содержать поле поиска, куда можно ввести имя врача, департамент или название конкретной процедуры.(Я думаю убрать отсюда поиск по департаментам т.к. на странице уже будет список департаментов) Помимо поля поиска, страница также должна содержать список департаментов, когда пользователь нажимает на определенный департамент, его перенаправляет на страницу где перечислены все врачи данного департамента. Каждый врач отображается следующим образом: имя, фамилия, фотография, список проводимых процедур и доступные слоты для записи. 
Выбрав врача, дату-время, и процедуру пациента просят подтвердить его выбор и после подтверждения запрос отправляется администратору. 
 
При вводе в поле поиска должны выходить поисковые подсказки. 
Пагинация при отображении списка врачей. 
 
Главная страница  
Главная страница должна содержать разделы: О нас, Услуги, Войти в аккаунт, Контакты 
Строку поиска по странице  
После входа в аккаунт пользователя перебрасывает на его личную страницу 
 
У каждой услуги есть цена и список противопоказаний 
Каждый департамент должен предоставить минимум 4 услуги 
 
Отчет 
Общее количество проведенных процедур, прибыль, число пациентов за определенный временной промежуток 
Количество пациентов на каждой процедуре, сколько пациентов пользовались услугами каждого департамента
